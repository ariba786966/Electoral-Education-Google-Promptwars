# Electoral-Education-Google-Promptwars

## Vision Statement: To empower every citizen with real-time, localized, and non-partisan electoral information, bridging the gap between complex election schedules and voter participation through an intuitive digital interface.

## Tech Stack
1. frontend - react.js
2. backend - node.js
3. containerization - docker
4. cloud deployment - google cloud run

## Features
1. An AI assistant
2. PWA (Progressive Web App) built keeping Gen-Z in mind
3. User can login using aadhaar and password. Ensure safety and security.
4. Voice enabled responses in  English
5. Responsive design for mobile, tablet and desktop
6. Offline access to critical information
7. Light colors used to give a soft and calm experience
